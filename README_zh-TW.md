# OpenClaw Android Dashboard

[English](README.md) | **繁體中文**

---

**OpenClaw Android Dashboard** 是一個原生的 Android 應用程式，專為在行動裝置上管理和監控您的 OpenClaw 實例而設計。

## 使用方式

1. **安裝**：下載並在您的 Android 裝置上安裝 APK。
2. **設定**：
   - 輸入您的 **Dashboard URL**
   - 輸入您的 **Gateway Token**
3. **連線**：點擊「連線」以建立與 OpenClaw 實例的安全連線。
4. **管理**：開始與您的代理聊天、查看活躍對話，並監控您的基礎設施！
