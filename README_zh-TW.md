# OpenClaw Android Dashboard

[English](README.md) | **繁體中文**

---

**OpenClaw Android Dashboard** 是一個原生的 Android 應用程式，專為在行動裝置上管理和監控您的 OpenClaw 實例而設計。

## 功能特色

- **無縫連接**：輕鬆連接到您的 OpenClaw Web Dashboard。
- **安全認證**：支援 Hugging Face Private Space 認證，確保存取安全。
- **對話管理**：查看和管理活躍的對話，並即時監控 Token 使用統計。
- **即時聊天介面**：透過響應式聊天介面直接與您的 AI 代理互動。
- **實例監控**：隨時掌握已連線裝置及其狀態。
- **現代化介面**：使用 Jetpack Compose 和 Material 3 打造，提供美觀、動態且流暢的使用者體驗。

## 使用方式

1. **安裝**：下載並在您的 Android 裝置上安裝 APK。
2. **設定**：
   - 輸入您的 **Dashboard URL**
   - 輸入您的 **Gateway Token**
3. **連線**：點擊「連線」以建立與 OpenClaw 實例的安全連線。
4. **管理**：開始與您的代理聊天、查看活躍對話，並監控您的基礎設施！
